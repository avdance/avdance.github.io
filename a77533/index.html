  • autotools (autoconf, automake)

可以通过homebrew自行安装:

编译安装

git clone https://github.com/man-pages-zh/manpages-zh #下载文件
cd manpages-zh
autoreconf –install –force #构建configure文件
./configure
make
make install #需要超级用户权限
